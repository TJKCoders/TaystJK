# Vulkan Renderer

-   Enable with `cl_renderer rd-vulkan; vid_restart`
-   `r_fbo 1; r_bloom 1` to enable bloom, adjust with `r_bloom_intensity` and `r_bloom_threshold`
-   See more info [here](https://github.com/ec-/Quake3e#vulkan-renderer)
-   From JKSunny
